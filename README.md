# 100 Days of Python

A personal learning journey through Python, one day at a time.

## About

This repository tracks daily Python exercises, projects, and notes as part of the [100 Days of Code](https://www.100daysofcode.com/) challenge. Each day builds on the last, covering fundamentals through more advanced topics.

## Structure

```
100daysPython/
├── day-01/
├── day-02/
└── ...
```

Each day folder contains scripts, exercises, or mini-projects for that day's lesson.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Dkolomy/100-day-python.git
   cd 100-day-python
   ```

2. Create a virtual environment (recommended):
   ```bash
   python -m venv venv
   ```

3. Activate the virtual environment:
   - **Windows:** `venv\Scripts\activate`
   - **macOS/Linux:** `source venv/bin/activate`

4. Install dependencies as needed for each day's project.

## Requirements

- Python 3.10+
